Instructions

If you see connection errors, follow these steps:
  1. Clear HSTS Cache:
    • Chrome/Edge: Go to chrome://net-internals/#hsts, enter 192.168.7.114 in "Delete domain security policies" and click Delete
    • Firefox: Close Firefox completely, then delete SiteSecurityServiceState.txt from your Firefox profile folder
    • Safari: Close Safari, then run in Terminal: defaults delete com.apple.Safari WebKitHSTSData
  2. Clear Browser Cache: Press Ctrl+Shift+Delete (or Cmd+Shift+Delete on Mac)
  3. Hard Refresh: Press Ctrl+Shift+R (or Cmd+Shift+R on Mac)
  4. Restart Browser: Close and reopen your browser completely
