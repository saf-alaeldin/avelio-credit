# Restart Backend Server

Restart only the backend development server.

## Steps

1. Kill any node process running on port 5001
2. Start the backend server from `avelio-backend` directory
3. Confirm it's running on http://localhost:5001

## Command

```bash
cd "D:\avelio-credit\avelio-backend" && npm start
```

Run in background and verify the server starts successfully.
