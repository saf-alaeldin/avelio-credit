# Restart Frontend Server

Restart only the frontend development server.

## Steps

1. Kill any node process running on port 3000
2. Start the frontend server from `avelio-frontend` directory
3. Confirm it's running on http://localhost:3000

## Command

```bash
cd "D:\avelio-credit\avelio-frontend" && npm start
```

Run in background and verify the server starts successfully.
